# Project Title

- Marketplace

## Tech Stack

- React & React Native
- NestJS
- MongoDB
- Docker
- Kubernetes

## Project installation

Clone the project

```bash
  git clone https://github.com/Los-quatros/challenge-s2-mono.git
```

Go to the project directory

```bash
  cd challenge-s2-mono
```

## Installation

Run all services

```bash
  ./start_all.sh
```

Run each service by one

```bash
  cd <folder-name>
  docker compose up -d
```

Stop all services

```bash
  ./stop_all.sh
```

Stop each service by one

```bash
  cd <folder-name>
  docker compose down --remove-orphans --volumes --timeout 0
```
